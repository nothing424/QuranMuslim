# QuranMuslim

Aplikasi Quran modern, minimalis, dan nyaman dibaca.

## Fitur
- Daftar Surah
- Ayat lengkap (Arab, Latin, Arti)
- Audio per ayat
- Auto next ayat
- Dark mode
- Search surah
